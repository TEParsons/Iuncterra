Kodama

Kodama are benevolent fey which manifest as pinpoints of light refracted through tree leaves, most commonly found in the forests of T'Savo. They have been known to use dancing lights guide travellers away from danger.

Statblock (5e)

Kodama

Fey rhea

Idle effect

If known, increases the number of lights created by the cantrip Dancing Lights by 1. If not known, grants the cantrip.

Activation effect

Cast Faerie Fire for free.

Summoning effect

Cast Hypnotic Pattern for free.

Summoning with more kodamae increases the Wisdom save DC by 2.
